## Install

[npm][]:

```sh
npm install timegrify
```

[yarn][]:

```sh
yarn add timegrify
```


## Usage

```js
const Timegrify = require('timegrify');

const timegrify = new Timegrify();

console.log(timegrify.renderName());
// script
```
